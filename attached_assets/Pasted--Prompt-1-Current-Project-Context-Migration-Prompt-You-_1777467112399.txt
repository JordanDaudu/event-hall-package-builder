Current project overview:
- Project name: **Event Hall Package Builder**
- Purpose: A full-stack event hall package/quote system.
- Original MVP idea: customers build an event package by choosing an event type, guest count, and upgrades. The frontend shows a live estimate, but the backend calculates the final price and saves the quote.
- Admins can view quotes, update quote statuses, manage upgrades, and view analytics.
